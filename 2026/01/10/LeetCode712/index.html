# 1. Intuition

This is a relatively simple DP question using LCS.

# 2. Approach

First, we need to understand what LCS is.

# LCS(Longest Common Subsequence)

# What is Longest Common Subsequence

If I give you two strings like "sea", "eat"
What is a longest Subsequence they both have?
Ans : "ea"

# Why don't we use Brute-force

If we use Brute-force Time Complexity will be $$O(2^N · 2^M)$$
because we need to enumerate all subsequences of both strings.

# How LCS work

First, We create a 2D vector(or array) size [N + 1][M + 1]
e.x. "sea" and "eat"
“X” means NULL, and we initialize the 0th index with 0.

And we follow this logic

if (s1[i - 1] == s2[j - 1]) {
    dp[i][j] = max(dp[i - 1][j - 1] + s1[i - 1], dp[i][j]);
} else {
    dp[i][j] = max(dp[i][j - 1], dp[i - 1][j]);
}

The final DP table will look like this.

# Using LCS in this problem

LCS vector's value meaning current the longest length we have
we change value for current the max cost we don't need to pay
e.x.

and we get two strings cost s1_sum, s2_sum
In the end, return s2_sum + s1_sum - dp[N][M] * 2;

# 3. Complexity

N -> s1.length, M -> s2.length

  • Time complexity: $$O(N*M)$$
  • Space complexity: $$O(N*M + 2)$$

# 4. Code

class Solution {
public:
    int minimumDeleteSum(string s1, string s2) {
        int s1_sum = 0, s2_sum = 0;
        for (char c:s1) {
            s1_sum += c;
        }

        for (char c:s2) {
            s2_sum += c;
        }
        
        const int N = s1.size(), M = s2.size();
        vector<vector<int>> dp(N + 1, vector<int>(M + 1, 0));
        dp[0][0] = 0;

        for (int i = 1; i <= N; i++) {
            for (int j = 1; j <= M; j++) {
                if (s1[i - 1] == s2[j - 1]) {
                    dp[i][j] = max(dp[i - 1][j - 1] + s1[i - 1], dp[i][j]);
                } else {
                    dp[i][j] = max(dp[i][j - 1], dp[i - 1][j]);
                }
                // cout << dp[i][j] << " ";
            }
            // cout << endl;
        }
        return s2_sum + s1_sum - dp[N][M] * 2;
    }
};
