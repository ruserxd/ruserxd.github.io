# 學習

# fill 函式

Data dp[1000][501];
fill(dp[0], dp[0]+(k+1), Data(0, -x0, x0));

意思就是把指定範圍的 value 換成相同的

Data dp[1000][501];
for (int i = 0; i <= k; i++) {
    dp[0][i] = Data(0, -x0, x0);
}

為何不單純用迴圈就好?

  • 更簡潔的程式碼
  • 屬於 C++ STL 函式，速度會比較快

# 題解

Source Code

struct Data{
    long long profit=0, buy=0, sell=0;
    Data(long long profit=0, long long buy=0, long long sell=0):
        profit(profit), buy(buy), sell(sell) {}
};
Data dp[1000][501];
class Solution {
public:
    static long long maximumProfit(vector<int>& prices, int k) {
        const int x0=prices[0];
        const int n=prices.size();
        fill(dp[0], dp[0]+(k+1), Data(0, -x0, x0));
        for(int i=1; i<n; i++){
            const int x=prices[i];
            for(int t=1; t<=k; t++){
                auto [p, b, s]=dp[i-1][t];
                const long long prevP=dp[i-1][t-1].profit;
                p=max(p, max(b+x, s-x));
                b=max(b, prevP-x);
                s=max(s, prevP+x);
                dp[i][t]=Data(p, b, s);
            }
        }
        return dp[n-1][k].profit;
    }
};

  1. 首先用 fill 把第 0 天的資料預設為 (第 0 天資本額為 0)

    profit = 0 -> 沒賺錢
buy = -prices[0] -> 買入股票所以為負數
sell = prices[0] -> 賣出股票

  2. 第一層迴圈 (每一天 i)
    當天的股票價格

    const int x=prices[i];

  3. 第二層迴圈 (交易次數 t)
    取出前一天交易次數 t 的 Data

    auto [p, b, s] = dp[i-1][t];

    取出前一天交易次數 t - 1 的獲利 (計算前天的獲利+這次的買賣可得)

    const long long prevP=dp[i-1][t-1].profit;

    更新當前第 i 天 交易次數 t 的 dp 值

    p=max(p, max(b+x, s-x));
b=max(b, prevP-x);
s=max(s, prevP+x);
