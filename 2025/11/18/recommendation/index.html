# 推甄看法

先說：「能多丟幾間就多丟幾間，畢竟沒人能預測結果」
以下為我的看法
校名 >>>>> 在校成績 >> 校外經歷
先說校名為何重要，從私立學校推甄上頂大難度也是真的很高。1%就能上，但1%也就250個抽2個，無法保證一定就是你。
自己人也傾向保自己人?陽交好像沒有XD
再來是在校成績，這部分是真的很重要，上述的部分已經沒辦法改變，除非重考、轉學考。
能快速從一份履歷中篩選，校名就是最快的，你說做過多厲害的專案，只要沒過他們要的標準，就 out。
最後是經歷，其實經歷也能幫助到蠻多的，像是陽交就有蠻多實務組，還有清大、成大也有正所乙組，我覺得有很不錯的大型專案、開源等經歷也能幫助到推甄，但就要真的很強。

# 心得

很幸運，上上禮拜推甄正取陽明交通大學資訊聯招甲組。
搭著通往新竹的客運，準備完成報到手續，順道書寫這三年的心得。
一路從中正高中社會組，考到逢甲資工，再推到交大資工所。
高中時根本沒想過以後會讀資工系，滿腦子想的都是課業外的事。上電腦課時，都刻意坐在最後一排胡搞瞎搞，伏地挺身之類的XD，程式是甚麼感覺就很無聊，倒不如多開幾把遊戲。高三突然一個轉念想當軟體工程師，就跑來逢甲資工就讀。
讀著讀著也突然覺得很有興趣，除了課堂上的也讀了軟體相關的書，心想怎麼會這麼有趣，因此大一成績就不錯，不知為何就也想繼續讀碩士，發下豪言:「沒上清大不改名」，沒人相信，我也不信。
推甄很注重成績，因此考的不是很理想時，就覺得讀了那麼久，怎麼那麼笨，是哪裡不夠努力，哪裡做錯了。深夜也時常失眠，怕錯過一堂課、一科考試，會不會錯過了一切都沒了，檢定停滯整整一年半，不停省思是否哪裡出問題，還是單純我太沒用。所以也看了不少心靈雞湯，調適心態的方式也越來越好?
讀書讀到無聊時，就會思考考高分、衝高排名、比競賽、寫計畫的意義在哪，究竟為何努力，是熱忱嗎，還是盲目地在做。未來究竟會去哪?上了好研究所意義又是甚麼?未來難道只能是一套標準答案，
考上好研究所→進科技廠輪班→一輩子的工程師→存下些許的錢，買下一棟房、一台車、偶爾出國走走→退休。
想了許多與考試無關的事，但想了想，還是想拿到”頂大入場卷”，畢竟就很帥咩，讀書的動力又加滿了，本已闔上的書本又翻開，繼續閱讀下篇章節，心想等上了再來想吧。
因此實驗室就沒找事多的，想要除了研究(找bug)以外花些時間持續探索未來藍圖。

因為你們的出現，點燃了我生命一絲絲的火光。
謝謝自己，縱使不知道當下的意義在哪，還是盲目地繼續下去。
祝各位推甄也順利 Lucky Vicky~
